                                                PROJETO JAVA - CADASTRAR LIVRO


O código foi criado com o intúito de cadastrar livros, contendo: // 1 - Cadastrar livro // 2 - Listar livros // 3 - Editar livro // 4 - Excluir livro // 5 - Buscar livro // 0 - Sair


COMO EXECUTAR O SISTEMA

Ao iniciar, será exibido um menu com as opções de cadastro, listagem, edição, exclusão e busca de livros. O usuário deve digitar o número correspondente à operação desejada e pressionar Enter.

Para cadastrar um livro, são solicitados o título, autor, gênero e ano de publicação. Os livros cadastrados ficam armazenados em uma lista durante a execução do programa.

A opção de listar exibe todos os livros cadastrados. A opção de editar permite alterar os dados de um livro por meio do seu ID. Já a exclusão remove um livro da lista utilizando o ID informado.

Também é possível buscar um livro pelo título. O sistema procura o livro na lista e exibe suas informações caso seja encontrado.

O programa permanece em funcionamento até que o usuário escolha a opção 0 - Sair, encerrando a execução do sistema.
